## Installation &

### Setup

1. Clone the repository:

    ```bash
    git clone git@github.com:CodencoIT/triplio-b2b-frontend.git
    cd triplio-b2b-frontend

    ```

2. Install dependencies:

    ```bash
    npm i

    ```

3. Create a .env file in the root directory and add your environment variables:
    ```bash
    NEXT_PUBLIC_API_BASE_URL=
    NEXT_PUBLIC_BANNER_IMAGE=https://flight24-bkt.s3.amazonaws.com/system/banner.png
   


    ```
4. Start the application:
    ```bash
    npm run dev
    ```
